Introduction

The everydayhero API

This describes the API resources that make up the everydayhero Fundraiser API v2, and everydayhero Passport API v1 (OAuth2). If you have any problems or requests please email api@everydayhero.com.

Most functionality is available via OAuth endpoints, which can be accessed with OAuth Integration. However, some resources will require additional credentials (see Authentication) you will need to contact the above email for credentials.
