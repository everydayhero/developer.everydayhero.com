Charities

List all Charities (public) 

GET https://everydayhero.com/api/v2/charities

View demo in API console

Parameters

campaign_ids : optional string
A comma-separated list of Campaign IDs, only Charities participating in the specified Campaigns will be retrieved.

Pagination

For more information about pagination, please see the pagination overview.

limit : optional integer
The number of items per page.

page : optional integer
The page number to return.

Example

https://everydayhero.com/api/v2/charities.json?campaign_ids=au-1,au-2,au-3

Response

{
  "charities": [
    {
      "id": "au-1",
      "name": "Starlight",
      "slug": "starlight",
      "country_code": "au",
      "description": null,
      "gift_aid": false,
      "logo_url": "http://"
    }
  ]
}

View a single Charity (public) 

GET https://everydayhero.com/api/v2/charities/:id

View demo in API console

Example

https://everydayhero.com/api/v2/charities/au-1.json

Response
