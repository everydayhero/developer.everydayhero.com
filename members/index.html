Members

Add a Team Member (requires authentication)

Add an existing page to an existing team.

POST https://everydayhero.com/api/v2/teams/:id/members

Payload

individual_page_id : required integer
The id of the page you want to join a team with.

Example

{
  "individual_page_id": 1
}

Response

{
  "team": {
    "id": 1,
    "uid": 1,
    "name": "Team Example"
  }
}

Remove a Team Member (requires authentication)

Remove a member from an existing team, the leader can not be the team leader. The page id must be a member of a campaign you are authenticated with.

DELETE https://everydayhero.com/api/v2/teams/:team_id/members/:page_id

Response

204 no content, for success

404 for invalid team or page
